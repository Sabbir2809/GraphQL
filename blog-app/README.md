# Blog App

## Requirements:

1. user can article publish, update and delete
1. user can view article

## Schema:

1. Post
   - id
   - title
   - content
   - authorId
   - published
   - createdAt
   - updatedAt
1. User
   - id
   - name
   - email
   - password
   - Profile
   - createdAt
   - updatedAt
1. Profile
   - id
   - bio
   - userId
   - createdAt
   - updatedAt

## Technology Stack:

1. TypeScript
1. GraphQL
1. Prisma
